# 클린매칭 - 청소업체 테스트 계정 목록
> 이 파일은 개발/테스트용입니다. 실제 운영 시 삭제하세요.

> ⚠️ **App 통합 시 주의**: 이메일은 users 테이블에서 UNIQUE 제약이라 `DataInitializer`가 `existsByEmail`로 중복 생성을 막아준다.
> 나중에 `app/` 코드에 별도 테스트 시드 데이터가 있다면, 병합 전에 여기 이메일 목록과 겹치는지 반드시 확인할 것 — 같은 이메일을 다른 내용으로 다시 만들려고 하면 에러 없이 조용히 무시되고 먼저 만들어진 계정이 그대로 남는다.
